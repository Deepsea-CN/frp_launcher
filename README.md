# frp_launcher

frp启动器，GUI管理你的frpc客户端

使用frp的xtcp,stcp进行内网穿透时需要客户端支持，本程序旨在方便优雅的启动frp客户端

未来功能：\
支持配置加密 \
一键分发客户端配置 \
服务端frps支持 \
支持导入和导出配置文件 

## #GUI版本

可以增删改查多个配置文件，管理frp启停

### 功能点：

表单式添加配置

图形化查看和选择配置文件

支持图形化启动和停止 frpc 服务。

实时查看日志。

### 使用说明：

添加配置：按需填入服务器和visitor信息（为保证安全，强制要求使用鉴权方式token）

修改配置：实时查看配置文件，可进行修改

删除配置：删除选中的配置文件

启动frp：选择配置文件后点击一键启动frp

停止frp：一键停止frp

配置列表：实时查看和选择配置文件

实时日志：实时打印日志


## #配置生成工具（未来功能）
